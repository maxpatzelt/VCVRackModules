# LFO Plugin

## Overview
LFO is a VCV Rack plugin that provides a Low-Frequency Oscillator (LFO) module for generating various waveforms for modulation purposes.

## Installation
1. Clone the repository or download the source code.
2. Navigate to the project directory.
3. Run the `make` command to build the plugin.
4. Copy the resulting plugin folder to your VCV Rack plugins directory.

## Usage
- **LFO Module**: 
  - Connect the output to a modulation destination.
  - Adjust the frequency knob to control the oscillation speed.
  - Use the sine wave output for smooth modulation signals.

## Module
- **LFO**
  - Description: Generates low-frequency waveforms for modulation.
  - Tags: LFO, Oscillator.

## Dependencies
This plugin may require additional libraries or modules. Please refer to the `dep` directory for any external dependencies.
